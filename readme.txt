== Description ==
Allows Wordpress to utilize the Cash Music (CashMusic.org) elements and assets found in the Cash Music System 

Allows the use of a simple WordPress short code to include the neat Cash Music elements like Email Collection, Tour Dates, E-Cards, Social Feeds and Logins. It allows you to sell your music or digital downloads using the Cash Music Commerce system (when Cash Music finishes it).
Check out CashMusic.org for info and demos of the system.

== Installation ==
1. Download the plugin.
2. Upload the plugin to your wp-content/plugins directory.
3. Activate the plugin.
4. Fill in the settings with information about the Cash Music installation
